# Danh mục bài tập
# Bài 1: Kiểm tra cấu hình máy tính
Xác định các thông số mạng của máy tính đang sử dụng:

IP address

Subnet mask

Gateway

DNS server

# Bài 2: Lý thuyết mạng cơ bản
Trình bày chức năng của hai thành phần quan trọng trong mạng:

Chức năng của Gateway: Điểm ra vào của một mạng, giúp kết nối các mạng khác giao thức với nhau.

Chức năng của DNS Server: Hệ thống phân giải tên miền, giúp chuyển đổi tên miền (như https://www.google.com/search?q=google.com) sang địa chỉ IP.

# Bài 3: Kiểm tra kết nối với lệnh tracert

Sử dụng lệnh tracert trên Windows để liệt kê các nút trung gian khi kết nối tới:


a) www.ctu.edu.vn 


b) gmail.com 


c) IP máy cùng phòng thực hành 


d) IP máy khác phòng thực hành (trong mạng CICT) 

# Bài 4: Mô phỏng mạng với Cisco Packet Tracer

Vẽ sơ đồ kết nối cho 2 phòng máy (mỗi phòng 8 PCs) sử dụng các thiết bị: Hub/Switch, Router, Modem....
